## 🚀 Proyectos
 
### [AlgoQuePedir](https://github.com/algo3-unsam/algo-que-pedir-2025-grupo5) — FullStack tipo PedidosYa ✅
Aplicación fullstack con gestión de productos, órdenes y usuarios. Frontend SPA (React + Svelte) y backend REST containerizados con Docker Compose. Proyecto grupal académico con flujo colaborativo en GitHub.
 
`React` `Svelte` `TypeScript` `Tailwind CSS` `Kotlin` `Spring Boot` `Docker Compose`
 
---
 
### [BookLibre](https://github.com/phm-unsam/frontend-2026-grupo-12) — Sistema de alquiler de libros ✅
Sistema de reserva y alquiler de libros con gestión de disponibilidad y usuarios. Diseño basado en Chakra UI. Entorno completamente dockerizado.
 
`React` `TypeScript` `Chakra UI` `Kotlin` `Spring Boot` `Docker Compose`
 
---
 
### [StudyQuest](https://github.com/LorenGrz/StudyQuest) — Plataforma de estudio con IA 🚧
Plataforma con sistema de matching de usuarios e **IA integrada** para convertir archivos PDF y DOCX en quizzes interactivos. Backend modular con NestJS y PostgreSQL.
 
`React` `TypeScript` `Tailwind CSS` `NestJS` `PostgreSQL` `Docker Compose`
